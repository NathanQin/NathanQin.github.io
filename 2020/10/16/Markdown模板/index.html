如果你想把一些文字、內容隱藏起來，並提供按鈕讓用户點擊顯示。可以使用這個標籤外掛。

Inline

inline 在文本里面添加按鈕隱藏內容，只限文字

( content 不能包含英文逗號，可用 ‚)

content

content: 文本內容

display: 按鈕顯示的文字 (可選)

bg: 按鈕的背景顏色 (可選)

color: 按鈕文字的顏色 (可選)

例：

哪個英文字母最酷？ 因為西裝褲(C裝酷)

門裏站着一個人? 閃

Block

block 獨立的 block 隱藏內容，可以隱藏很多內容，包括圖片，代碼塊等等

( display 不能包含英文逗號，可用 ‚)

content

Toggle

如果你需要展示的內容太多，可以把它隱藏在收縮框裏，需要時再把它展開。

( display 不能包含英文逗號，可用 ‚)

display

Button

[text]

[url] : 鏈接

[text] : 按鈕文字

[icon] : [可選] 圖標

[color] : [可選] 按鈕背景顔色(默認style時）

​ 按鈕字體和邊框顔色(outline時)

​ default/blue/pink/red/purple/orange/green

[style] : [可選] 按鈕樣式 默認實心

​ outline/留空

​ block/留空
